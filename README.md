# vue_first

Vue cli 를 일단 설치
```shell
npm install -g @vue/cli
```

Vetur 설치
![vetur](./md/vetur.png)
Vue 개발 하면서 가장 많이 쓰는 플러그인 이라고 함
- Syntax Highligh
- Code Formatting
등 여러가지 기능을 수행

```shell 
vue create [project name]
```
으로 vue 실행 하면 최초 페이지 생성

``` shell 
npm run serve
```
